Are u Cool?:

Link: https://repl.it/@KawaiiOuran/Are-u-Cool

About: I made this python code during the summer as a funny pass time. Test how cool you are with either the game or the quiz. The game asks you yes or no questions to test your coolness, while the quiz puts you in scenarios and gives you decisions to choose from. When you are done with either it gives you your score.

Notes: You may either make a person before playing, OR you can use the dummy person already made named 'kelli'. Names are not case sensitive. Every time the code stops it loses all of your data.

Frog Simulator!

Repl.it Link: https://repl.it/@KawaiiOuran/Frog-Sim#main.py

GitHub Link: https://github.com/kelli-the-coder/Frog-Simulator

About: This was an on and off project of mine for a few months and I finally finished it in July! It's probably my biggest project yet. In this 2D frog sim you get to make your own frog and eat, fight, mate, and have offspring! I also used SQLite for the first time in this project, so you can save your frog for next time! You can play it on Repl.it but feel free to download the code and files from GitHub to run on an IDE for a better experience!

Notes: The game looks a little different in repl.it than it does in an IDE. The game does not save your data automatically you must click the save button on the main menu before stopping the code.
